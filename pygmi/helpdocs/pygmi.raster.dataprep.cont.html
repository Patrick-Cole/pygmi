Continuation

This is an upward and downward continuatione for Magnetic and Gravity data. Upward continuation is performed using teh standard FFT transform, while downward continuation used a Taylor expansion for extra stability.

The input is a raster dataset.

Options

  • Band to Continue - This should be magnetic or gravity data.
  • Continuation type - This can be upward or downward.
  • Continuation distance - a positive number denoting the continuation distance.
