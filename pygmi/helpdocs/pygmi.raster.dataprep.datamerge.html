Data Merge

This will allow datasets to be merged or mosaiced. The input is a directory which contains ONLY data to be merged. Two merging modes are possible.

If all files to be merged have the same number of bands, in the same order (band name is not checked) then a fast merge will result. This is the default. Alternatively, the user can merge files with different numbers of bands, and only bands with common names will be merged. This option is slower.

Options

  • Batch directory - location of files to be merged.
  • Input files have different numbers of bands, but common band labels for bands to merge - unchecked by default.
