TDEM 1D Inversion

TDEM inversion makes use of the SimPEG library (see references below). The inversion is 1D, so it requires as input a single measurement. Line data is imported using the Import Line Data tool in the Vector menu. The current TDEM tool is developed for EM data which has a VTEM waveform (i.e. VTEM and SkyTEM sensors).

Options

  • Line Number
  • Fid/Station Name
  • Bird Height - Height of the sensor.
  • Tx Area - Transmitter area
  • Tx Peak Time - Time when the input VTEM waveform is at its maximum.
  • Tx Off Time - Time when the waveform is completely off.
  • Data Channel Prefix - Prefix in ASCII data file, indicating a data channel.
  • Invert Station - used to invert the current station

References
