Text File Split

Text files can be split into a series of smaller files, either by number of files, bytes per file or lines per file.

Options

  • Split method - this can be Files, Lines or Bytes.
  • Number of files - editable using Files method.
  • Max lines per file - editable using Lines method.
  • Max bytes per file - editable using Bytes method.
  • Split all text files with same extension in current directory - will use specifications of original file on all subsequent files.
