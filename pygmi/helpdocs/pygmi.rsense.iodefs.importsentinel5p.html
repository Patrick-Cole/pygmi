Import Sentinel 5-P Data

Sentinel 5-P data is imported here, but unlike other imports, this converts data to a vector data format, from where it can be exported to a shapefile using the context menu. It accepts *.nc data. This import also allows the user to cut the data according to a bounding box.

Options

  • Product - Relevant product contained in the .nc file.
  • Minimum Longitude
  • Maximum Longitude
  • Minimum Latitude
  • Maximum Latitude
