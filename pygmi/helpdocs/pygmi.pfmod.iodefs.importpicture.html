New Custom Profile

The profile editor allows for the creation and display custom profiles. A series of coordinates for the custom profile are entered. The user is allowed load in a picture for superimposition along the section of the profile.

Note that if the custom profile extends beyond the model, it will be truncated. However, in the event a picture is loaded, the original coordinates are assumed to belong to the picture and will be assigned to it appropriately, resulting in a truncation of the image in the model.

Options

  • Get coordinates from last profile - Convenience function used to auto fill coordinates.
  • Load Picture (optional) - An optional picture importer.
