Data Layer Stack and Resampling

This will allow datasets to be stacked and resampled to a common cell size. It can also be used for a single dataset to be resampled. The resulting rows and columns of the new grid are displayed.

Options

  • Cell size - This represents the size of a square raster grid cell, in the units of the grid (normally meters).
  • Common mask for all bands - This option forces the nodata areas to be the same on all bands.
