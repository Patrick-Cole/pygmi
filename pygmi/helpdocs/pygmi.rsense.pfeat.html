Hyperspectral Feature Processing

This module takes as input hyperspectral data with wavelengths, and allows for a standard set of feature definitions to be calculated. Currently this includes mica, smectite, kaolin, chlorine, epidote and ferrous iron. Once a product is selected, the details of the various datasets which make up the calculation are shown on a table, including feature name, filter, threshold and description. The user is able to alter the respective dataset threshold.

Options

  • Product - includes mica, spectate, kaolin, chlorine, epidote and ferrous iron.
  • Details - including feature name, filter, threshold and description. The user is able to alter the respective dataset threshold. .
