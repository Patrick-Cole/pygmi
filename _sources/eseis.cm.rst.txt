Exploration Sesimics: Context Menu
==================================
The seismics menus are available for modules which have output seismic data. Output data is only available on green modules. To access the context menu, simply right-click a green module.

Show Graphs
-----------
SEG-Y data is dsiplayed using this tool. It can then be exported from here as a graphical image, for use in the modelling module, if necessary.

.. image:: _images/eseisdisp.png

Export SEG-Y
------------
Traces can be exported to SEG-Y data if needed.
